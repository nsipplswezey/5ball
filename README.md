# Local nodal
Nodal docs advise global install. I prefer local and using package.json scripts to use the package.

To access nodal use:
```
npm run nodal -- [arguments]
```

You can look in package.json to see the nodal script
```
"scripts":{
  "nodal" : "nodal"
  }
```
